🤖 Integracion con Inteligencia Artificial MCP

Este sistema expone un servidor MCP (Model Context Protocol) que permite a cualquier modelo de lenguaje consultar los datos del sensor en tiempo real, calcular promedios, detectar anomalias y generar analisis contextualizados.

URL del servicio MCP:
https://linnie-noncommemoratory-overforwardly.ngrok-free.dev/sse
Como conectar un LLM al servicio MCP:
1
Abrir Claude.ai e ir a Personalizar → Conectores → Agregar conector personalizado
2
En Nombre escribir: estacion-meteorologica
3
En URL del servidor MCP remoto pegar la URL del servicio mostrada arriba y click en Agregar
4
Preguntar al LLM, por ejemplo: ¿Cual es la temperatura actual? o ¿Hay alguna alerta meteorologica?
Herramientas disponibles: obtener_ultima_lectura, obtener_ultimas_lecturas, obtener_datos_grafico, obtener_resumen_estacion, detectar_alertas, datos_para_dashboard
